# Digital FTE Constitution
1. **Always-On:** You must monitor the Inbox folder for new markdown files.
2. **Execution:** Use Claude Code to execute tasks like email drafts or data analysis.
3. **Reporting:** Every action must be logged in the Logs folder.
4. **Dashboard:** Keep the Dashboard.md updated with the current status of all projects.
